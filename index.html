What is a Feistel Cipher?

Imagine you have a secret message, and you want to keep it safe from anyone who might try to read it. The Feistel Cipher is like a magic trick that scrambles your message in a way that only someone with the secret key can unscramble it.

Step-by-Step Explanation

1. Split the Message

First, take your message and split it into two parts. Let's use a simple example message: "HI" (we'll convert it to binary to make it easier to understand).

"H" in binary: 01001000
"I" in binary: 01001001

So, our message in binary is: 01001000 01001001

Split it into two parts:
Left (L) = 01001000
Right (R) = 01001001

2. Prepare for Rounds

We will use a key to scramble the message. Let's say our key is a simple binary number: 10101010.

3. Start the First Round

Round 1:

  • Take the Right half (R): 01001001
  • Combine it with the key using a special function (let's just XOR them for simplicity):
R = 01001001
Key = 10101010
XOR Result = 11100011

Now, XOR this result with the Left half (L):

L = 01001000
XOR Result = 11100011
New Right (R) = 10101011
  • Swap the halves:
    • New Left (L) = Old Right (R) = 01001001
    • New Right (R) = 10101011

4. Repeat for More Rounds

Let's do one more round to make it secure.

Round 2:

  • Take the new Right half (R): 10101011
  • Combine it with the key using XOR:
R = 10101011
Key = 10101010
XOR Result = 00000001

Now, XOR this result with the new Left half (L):

L = 01001001
XOR Result = 00000001
New Right (R) = 01001000
  • Swap the halves again:
    • New Left (L) = Old Right (R) = 10101011
    • New Right (R) = 01001000

5. Finish Up

After the rounds, put the final Left and Right halves together. This is your scrambled message (ciphertext):

Ciphertext: 10101011 01001000

6. Decryption

To decrypt, you do the same steps in reverse order using the same key. This will give you back the original message.

Summary

  • Message: "HI" (01001000 01001001)
  • Key: 10101010
  • Ciphertext after 2 rounds: 10101011 01001000

The Feistel Cipher is like a series of magic tricks that mix up your message using a key. Only someone with the same key can reverse the tricks and read the original message.

Was ist eine Feistel-Chiffre?

Stell dir vor, du hast eine geheime Nachricht. Du willst sie vor anderen geheim halten. Die Feistel-Chiffre ist wie ein Zaubertrick. Sie mischt deine Nachricht so, dass nur jemand mit dem richtigen Schlüssel sie wieder lesen kann.

Erklärung Schritt für Schritt

1. Teile die Nachricht

Nimm zuerst deine Nachricht und teile sie in zwei Hälften. Wir nehmen als Beispiel "HI" (wir machen daraus Zahlen, damit es einfacher zu verstehen ist).

"H" als Zahlen: 01001000
"I" als Zahlen: 01001001

Unsere Nachricht als Zahlen ist also: 01001000 01001001

Teile sie in zwei Hälften:
Links (L) = 01001000
Rechts (R) = 01001001

2. Bereite die Runden vor

Wir brauchen einen Schlüssel, um die Nachricht zu mischen. Unser Schlüssel ist einfach: 10101010.

3. Starte die erste Runde

Runde 1:

  • Nimm die rechte Hälfte (R): 01001001
  • Mische sie mit dem Schlüssel (wir benutzen dafür XOR):
R = 01001001
Schlüssel = 10101010
Ergebnis = 11100011

Jetzt mische dieses Ergebnis mit der linken Hälfte (L):

L = 01001000
Ergebnis = 11100011
Neues Rechts (R) = 10101011
  • Tausche die Hälften:
    • Neues Links (L) = Altes Rechts (R) = 01001001
    • Neues Rechts (R) = 10101011

4. Mache noch eine Runde

Wir machen noch eine Runde, damit es sicherer ist.

Runde 2:

  • Nimm die neue rechte Hälfte (R): 10101011
  • Mische sie mit dem Schlüssel:
R = 10101011
Schlüssel = 10101010
Ergebnis = 00000001

Jetzt mische dieses Ergebnis mit der neuen linken Hälfte (L):

L = 01001001
Ergebnis = 00000001
Neues Rechts (R) = 01001000
  • Tausche die Hälften wieder:
    • Neues Links (L) = Altes Rechts (R) = 10101011
    • Neues Rechts (R) = 01001000

5. Fertig

Nach den Runden fügst du die letzten linken und rechten Hälften zusammen. Das ist deine gemischte Nachricht:

Gemischte Nachricht: 10101011 01001000

6. Entschlüsseln

Um die Nachricht wieder zu lesen, machst du alles rückwärts mit dem gleichen Schlüssel. So bekommst du die ursprüngliche Nachricht zurück.

Zusammenfassung

  • Nachricht: "HI" (01001000 01001001)
  • Schlüssel: 10101010
  • Gemischte Nachricht nach 2 Runden: 10101011 01001000

Die Feistel-Chiffre ist wie eine Reihe von Zaubertricks. Sie mischt deine Nachricht mit einem Schlüssel. Nur jemand mit dem gleichen Schlüssel kann die Tricks rückgängig machen und deine Nachricht lesen.
